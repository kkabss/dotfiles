# Gentoo Dotfiles

My dotfiles for my xfce + i3 setup on gentoo.

## Custom aliases = 

The Custom Aliases that are set in i3 config file:

"walset" = set wallpaper with feh and colors with pywal

## Installation

To install my dotfiles, first clone the repo, then run:

 ```bash
   cd dotfiles
   ./install.sh
 ```
